## ✨ ویژگی‌های کلیدی

این پروژه علاوه بر دانلود موسیقی، از یوتیوب نیز پشتیبانی کرده و دارای یک سیستم قدرتمند برای مدیریت تبلیغات و اسپانسرهاست.

| ویژگی | توضیحات |
| :--- | :--- |
| 🌐 **دانلود از پلتفرم‌های مختلف** | پشتیبانی جامع از **YouTube**، **Spotify**، **SoundCloud** و **Deezer**. |
| 🎥 **پردازش هوشمند رسانه** | استخراج صدا از ویدئوها، انتخاب کیفیت‌های مختلف استریم یوتیوب و پردازش با `MoviePy`. |
| 🏷️ **ویرایش متادیتا (ID3 Tags)** | تنظیم خودکار کاور آرت (Cover)، نام خواننده و آلبوم روی فایل‌های صوتی با استفاده از `eyeD3`. |
| 📢 **سیستم پیشرفته اسپانسرینگ** | قابلیت تعریف اسپانسر، وضعیت فعال/غیرفعال و نمایش خودکار پیام‌های اسپانسری به کاربران. |
| 🔒 **عضویت اجباری (Force Join)** | سیستم دقیق بررسی عضویت کاربران در کانال‌های معرفی‌شده پیش از ارائه خدمات. |
| ⏳ **مدیریت صف و وضعیت موقت** | استفاده از مدل‌های دیتابیس موقت (`TempYoutube`، `TempSpotify` و...) برای مدیریت بدون خطای دانلودهای همزمان. |
| 👥 **سطوح دسترسی و مدیریت** | جداسازی کاربران عادی و ادمین‌ها با فیلترهای اختصاصی و مدیریت کامل از طریق پنل جنگو. |

---

## 🛠️ تکنولوژی‌ها و ابزارهای استفاده شده

- **زبان برنامه‌نویسی:** Python 3.x
- **فریم‌ورک بک‌اند:** Django (برای ORM، مدیریت دیتابیس و پنل ادمین)
- **کتابخانه تلگرام:** Pyrogram / Pyromod
- **ابزارهای دانلود و پردازش:** Pytubefix / Pytube (برای یوتیوب)، sclib (ساندکلاد)، MoviePy (پردازش ویدئو) و eyeD3 (متادیتا)
- **دیتابیس:** SQLite (پیش‌فرض) / PostgreSQL
